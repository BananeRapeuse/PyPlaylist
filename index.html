FAQ - PyPlaylist

Q: What is PyPlaylist ?
A: PyPlaylist is a software codded in Python, it's a cool UI playlist manager for the songs you've downloaded on internet

Q: Which OSs does PyPlaylist compatible with ?
A: PyPlaylist is only compatible with Windows due to the mpv player that is an exe file.

Q: Does it affect my gaming perf ?
